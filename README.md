Kladr для SkeekS CMS
===================================

Информация о модуле
-------------------

База адресов для SkeekS CMS

Установка
------------

1) Добавить в `composer.json` файл проекта.

```
"skeeks/cms-kladr": "*"
```

2) Запуск миграций и необходимых установок.

```
php yii cms/update
```
